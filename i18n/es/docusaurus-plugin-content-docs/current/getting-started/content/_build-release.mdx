Ahora estás listo para crear una release de Velopack para tu aplicación. 

El `--packId` puede ser cualquier identificador único de aplicación que desees usar. Como debe ser único entre todas las aplicaciones, 
recomendamos incluir el nombre de tu empresa: `<CompanyName>.<AppName>`.

La opción `--mainExe` solo es necesaria si el nombre de tu ejecutable es diferente del `--packId` de tu aplicación.

Consulta la [referencia de la CLI](/reference/cli/) para más detalles sobre las opciones disponibles.

```cmd
vpk pack --packId YourAppId --packVersion 1.0.0 --packDir .\publish --mainExe yourMainApp.exe
```
