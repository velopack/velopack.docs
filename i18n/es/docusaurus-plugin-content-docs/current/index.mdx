# Introducción
Velopack es un framework de instalación y actualización automática para aplicaciones de escritorio multiplataforma. Es opinado, extremadamente fácil de usar y no requiere ninguna configuración. Con un solo comando puedes tener en marcha una aplicación instalable, y además es ultrarrápido para tus usuarios.

¡No dejes de visitarnos en [GitHub](https://github.com/velopack/velopack) y [únete a nuestro Discord](https://discord.gg/CjrCrNzd3F) para cualquier pregunta o soporte!

## Resumen
Para que tu aplicación aproveche al máximo Velopack, debes hacer 3 cosas:

<FancyStep step={1}>
### Integra el SDK y comprueba si hay actualizaciones
Tenemos bibliotecas cliente para varios [lenguajes a continuación](#language-support), o si aún no soportamos tu lenguaje puedes [**abrir una incidencia para solicitarlo**](https://github.com/velopack/velopack/issues). <br/>
[Leer más →](integrating/overview.mdx)
</FancyStep>

<FancyStep step={2}>
### Crea una release de Velopack
Nuestra herramienta de línea de comandos `vpk` puede ayudarte a crear paquetes de actualización e instaladores con un solo comando. <br/>
[Leer más →](packaging/overview.mdx)
</FancyStep>

<FancyStep step={3} noline>
### Sube tus releases a algún lugar
Puedes alojar las actualizaciones en cualquier lugar donde se puedan servir archivos estáticos, p. ej. almacenamiento de archivos en la nube, GitHub Releases y más. <br/>
[Leer más →](distributing/overview.mdx)
</FancyStep>
