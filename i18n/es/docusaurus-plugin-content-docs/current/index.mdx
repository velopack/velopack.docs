# Introducción
Velopack es un framework de instalación y actualización automática para aplicaciones de escritorio multiplataforma. Es opinado, extremadamente fácil de usar sin necesidad de configuración. Con un solo comando puedes tener una aplicación instalable en funcionamiento, y también es muy rápido para tus usuarios.

Asegúrate de visitarnos en [GitHub](https://github.com/velopack/velopack) y [únete a nuestro Discord](https://discord.gg/CjrCrNzd3F) para cualquier pregunta o soporte.

## Descripción general
Para permitir que tu aplicación aproveche al máximo Velopack, necesitas hacer 3 cosas:

<FancyStep step={1}>
### Integra el SDK y busca actualizaciones
Contamos con bibliotecas cliente para varios [idiomas a continuación](#language-support), o si aún no admitimos tu idioma puedes [**abrir un issue para solicitarlo**](https://github.com/velopack/velopack/issues). <br/>
[Leer más →](integrating/overview.mdx)
</FancyStep>

<FancyStep step={2}>
### Crea un release de Velopack
Nuestra herramienta de línea de comandos `vpk` puede ayudarte a crear paquetes de actualización e instaladores con un solo comando. <br/>
[Leer más →](packaging/overview.mdx)
</FancyStep>

<FancyStep step={3} noline>
### Sube tus releases a algún lugar
Puedes alojar actualizaciones en cualquier lugar donde se puedan servir archivos estáticos, por ejemplo, almacenamiento de archivos en la nube, GitHub Releases y más. <br/>
[Leer más →](distributing/overview.mdx)
</FancyStep>
