# Enlaces estáticos
<FlowLink /> proporciona enlaces estáticos para cada canal y versión. Estos enlaces pueden referenciarse desde tu propio sitio web para dirigir a los usuarios a descargar la última versión de una aplicación, o a descargar una versión específica. 

El más común será la última versión de un canal determinado. En la página de la lista de canales puedes hacer clic en el menú de puntos suspensivos en el lado derecho de la fila del canal y seleccionar "Copy latest setup link". Esto copiará a tu portapapeles un enlace al instalador de configuración más reciente de ese canal. Esta URL siempre apuntará a la última versión de ese canal, por lo que puedes usarla en tu propio sitio web o documentación.
![Enlace estático de la última versión de un canal](/images/latest-channel-link.png)

También puedes obtener un enlace directo a versiones específicas. En la página de detalles de la versión hay una sección de Release links con botones para copiar enlaces a instaladores de versiones específicas. 

Para casos de uso más avanzados, consulta los endpoints `download/*` en la sección Releases de [API](./api.mdx). La API ofrece funcionalidad adicional para obtener enlaces a artefactos específicos o a tipos de artefactos.
